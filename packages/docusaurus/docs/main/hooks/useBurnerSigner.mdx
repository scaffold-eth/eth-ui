Creates a burner signer/address, and provides ways of interacting with and updating the signer.

> Returns the burner signer/address.

## How to import

```typescript
import { useBurnerSigner } from 'eth-hooks';
```

## How to use

To get started:

```typescript
const burnerSigner = useBurnerSigner(localProvider): TBurnerSigner;
```
